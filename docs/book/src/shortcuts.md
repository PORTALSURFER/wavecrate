# Keyboard Shortcuts

Wavecrate shortcuts are focus-aware. A key can mean one thing in the waveform, another in the sample browser, and nothing while a text field owns keyboard input.

Press `Command-/` in the app to open the current context-aware shortcut overlay.

## Global

| Shortcut | Action |
| --- | --- |
| `Space` | Play or pause |
| `Shift-Space` | Replay from the last start |
| `Esc` | Close transient UI, stop playback, or clear selection |
| `Command-Z` | Undo |
| `Command-Shift-Z` or `Command-Y` | Redo |
| `Command-/` | Toggle shortcut help |

## Waveform

| Shortcut | Action |
| --- | --- |
| `E` | Extract selection |
| `Command-E` | Extract and trim selection |
| `C` | Crop selection |
| `D` | Trim selection |
| `R` | Reverse selection |
| `M` | Mute selection |
| `L` | Toggle loop playback |
| `Command-C` | Copy current waveform selection as an exported WAV clip |

## Sample Browser

| Shortcut | Action |
| --- | --- |
| `Up` / `Down` | Move selection |
| `Shift-Up` / `Shift-Down` | Extend selection |
| `Right` | Mark selected sample as Keep |
| `Left` | Mark selected sample as Trash |
| `Command-C` | Copy selected sample files |
| `N` | Normalize selected sample |
| `D` | Delete selected sample |
